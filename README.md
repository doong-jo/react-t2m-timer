This repository is my frontend boilerplate. Feel free to use this.

## Stack

- Typescript
- React (based CRA), styled-component
- Storybook
- Jest, react-testing-library, Cypress
- Eslint, Stylelint, Commitlint, Prettier, Editorconfig
